# ECONOMICS

## Lead value model

If Credex earns 15-20% margin on a $5,000 average enterprise credit deal:

- Average converted lead value = $750 to $1,000

## Funnel assumptions

- 1,000 audits completed -> 200 email captures (20%)
- 200 leads -> 30 consultations (15%)
- 30 consultations -> 8 purchases (27%)
- 8 * $750 average value = $6,000 from 1,000 audits

Organic distribution from HN/communities keeps top-of-funnel CAC low.

## $1M ARR thought experiment

At $750 value per converted purchase:

- Need ~1,333 purchases for $1M annualized value
- At the funnel above, need ~167,000 audit completions

This requires recurring SEO/distribution, not one-time launch spikes.
